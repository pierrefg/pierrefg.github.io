Research Projects

  • Thesis • Study of normality in a system instrumented with sensors,
    application to hydroelectricity production
    • 1INSA Lyon, LIRIS UMR 5205, Villeurbanne, France
    2Compagnie Nationale du Rhône, Lyon, France
    2020-now

    To assess the normality of a system, we first worked on ways to measure and visualize the existence of a function in a dataset (Does my data verify my theoretical model?). We conceived a web application called ADESIT [D1] to visualize the counterexample graph of a function and its associated indicators. Moreover, we focused on developing efficient techniques to compute the g3 indicator [R1], a powerful indicator which finds its use in function existence measurement, machine learning and database theory in general. All our implementations have been made available in a Python package: Fastg3. This work has been presented in [W3,W4,W5].
    In parallel, we also work on various aspects of hydropower production from a data perspective. Two main projects are currently under study: airgap monitoring in a turbine and hydraulic head loss analysis.

  • Master's Thesis • Deep-learning for automatic diatom identification with synthetic data
    • GeorgiaTech Lorraine-IRL 2958, GeorgiaTech-CNRS, Metz, France
    2019-2020

    As part of my Master's thesis for Georgia Tech and my end-of-studies R&D internship for INSA Lyon, our research work focuses on automating the identification and classification of diatoms (microalgaes) on microscope images using the latest advances in object detection and image classification. Notably, we propose a novel approach using synthetic images to reduce the number of hand-labelled real images needed for training. Most of this work is presented in my Master's Thesis report [MT] and has been presented in workshops [W1,W2].
